## New Features
- Weather Picker now has a gear button to jump straight to Weather settings in the Calendar Editor
- `timestampToDate()` API returns `ordinal`, `monthName`, and `intercalary` for calendars with intercalary months
- `dateToTimestamp()` API accepts `ordinal` as an alternative to `month`

## Bug Fixes
- Temperature of 0 no longer treated as empty in Weather Picker
- "Disable Weather FX" scene flag now works on day change
- Extreme wind speed reduced from 999 kph to a realistic 250 kph
- `CALENDAR_SWITCHED` hook now fires at startup
- Forecast wind speeds no longer re-randomize on calendar re-render
- Weather sounds no longer require FXMaster to be enabled
- "Disable Weather FX" no longer mutes weather sounds (use "Disable Weather Sound" instead)
- Weather sounds now use the Environment volume slider instead of Music
- Periodic seasons longer than a year now cycle correctly across multiple years
