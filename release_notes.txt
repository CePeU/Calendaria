## New Features
- [#477] Wind direction now uses meteorological convention — labels show where wind comes from, arrow and particles show where it blows
- [#467] Sun Dial closes when clicking outside it (when opened from the HUD dome)
- [#460] "Disable Weather Effects" per-client setting turns off HUD/Sun Dial particle animations while keeping sky, sun, moon, and stars
- [#473] `clearWeatherHistory()` API for managing weather history via macros
- `addCalendar()` API for programmatic calendar creation
- All 7 climate zone templates now have explicit weather definitions for all 4 seasons
